About Me

Welcome. I have always had a passion for helping people and for working with computers. My passion for people led me to pursue a Bachelor's degree in Psychology from The Ohio State University. After graduation, I developed an interest in coding and ended up completing courses in Software Development. Most recently, I joined the We Can Code IT coding bootcamp. This program teaches both back-end and front-end development using IntelliJ and Visual Studio Code as the main environments. The coding languages that the program focuses on are Java, HTML, CSS, and JavaScript. I find coding to be very rewarding since it gives me the ability to create programs that have the potential to make the lives of other people easier. I am excited to start a career in Software Development.
